# My Astro Website

github pagesで公開している私の自己紹介やメモを記録するHPのレポジトリです。

## プロジェクト概要

github pagesでは、私の自己紹介と技術的なメモ、作成したパーツ、ポートフォリオを載せています。

サイトURL: [プロフィールページ](https://hiroyuki9614.github.io/my_profile/)

## 技術スタック

- [Astro](https://astro.build/)
- HTML/CSS
- TypeScript
- tailwindCSS
