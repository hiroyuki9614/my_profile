## mdxとは

ざっくり、MarkdownとJSXを組み合わせたもの。

両者の説明については自分では追いつかないほど素晴らしい発信者がいるので省略。

\{}の中にJavaScriptを入れるとMarkdownの中でJavaScriptを使用できる。

## 実験

このタグを操作する。

これをJSXでクリック時にメッセージを表示させたい。

export const handleClick = () => {
 console.log('クリックされました！');
};

<a id='move' onClick={handleClick}>
 メッセージ
</a>
