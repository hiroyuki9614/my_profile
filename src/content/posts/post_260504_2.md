## Reactでエイリアス @から始まるパスを使う方法

階層が深くなってくると、import文で`../../../`のように相対パスが複雑になってしまいます。
Reactプロジェクトで、`@`から始まるエイリアスを使って、`src`ディレクトリを基準にしたパスを指定する方法です。

自分自身、毎回ハマっていて次回もハマりそうなのでメモしておきます。

### 環境

React v11.4.2
Vite v7.3.2
TypeScript v5.9.3

### Viteでの設定

Viteを使っている場合は、`vite-tsconfig-paths`というプラグインをインストールして、`vite.config.ts`に以下のように追加します。
>npm i -D vite-tsconfig-paths

### tsconfig.app.jsonでの設定

検索をすると、`tsconfig.json`に設定を追加するという記事が多かったですが、`tsconfig.app.json`に設定を追加する必要がありました。

`tsconfig.app.json`の`compilerOptions`に以下のように追加します。

```json
{
  "compilerOptions": {
 "baseUrl": "./src",
 "paths": {
   "@/*": ["*"]
 }
  }
}
```

これで、`@/components/Button`のように、`src/components/Button`をエイリアスで指定できるようになります。
